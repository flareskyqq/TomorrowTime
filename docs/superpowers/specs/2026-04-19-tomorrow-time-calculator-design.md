# 明天时间计算器 — 设计文档

## 1. 项目概述

**项目名称：** 时间计算器（TomorrowTime）
**项目类型：** 微信小程序
**核心功能：** 设置明天的某个时间点，计算从当前时间到该时间点经历的时长（小时+分钟），支持预设多个时间方便快速选择。
**目标用户：** 需要计算距离明天某个时间点还有多久的用户。

## 2. 技术选型

- **框架：** 原生微信小程序（不使用第三方框架）
- **样式：** WXSS + CSS Variables（蓝色主题）
- **数据存储：** wx.setStorage/wx.getStorage（本地）
- **无后端、无数据库**

## 3. UI/UX 设计

### 3.1 页面结构

单页面应用（所有功能在一个页面内），不涉及多页面路由跳转（"返回"按钮在有其他页面的场景下使用）。

### 3.3 颜色主题

- **主色调（蓝色）：** `#1A73E8`
- **次要色：** `#E8F0FE`（浅蓝背景）
- **文字色：** `#202124`（深灰）
- **边框/分割线：** `#DADCE0`
- **背景色：** `#FFFFFF`
- **次要按钮：** 白色背景 + 蓝色边框

### 3.4 组件细节

#### 预设时间列表
- 每项显示格式：`HH:mm`
- 选中状态：浅蓝色背景 `#E8F0FE` + 左侧蓝色指示条
- 左滑或长按显示"删除"按钮
- 空状态：显示"暂无预设时间，请点击下方添加"

#### 增加按钮
- 文字按钮，蓝色 `+` 图标
- 位置在列表下方

#### 时间选择器（弹窗形式）
- 底部弹窗
- 两个滚轮：小时（00-23）、分钟（00-59）
- 默认值：当前时间顺延1小时（如当前9:20则默认10:20）
- 标题栏：左侧"取消"，右侧"确定"
- 滚轮样式：选中项放大 + 蓝色高亮

#### 计算按钮
- 蓝色圆角矩形，宽 80%，高 48px，白色文字
- 点击后计算从"当前时间"到"明天选中时间点"的时长
- 未选中预设时间时，按钮置灰不可点击

#### 计算结果
- 文字居中显示，如："距离明天 08:30 还剩 10 小时 25 分钟"
- 若已过明天的设置时间（用户第二天才打开小程序），显示"已过 X 小时 Y 分钟"

#### 导航栏
- 使用微信原生导航栏，左上角默认显示"返回"按钮
- 页面标题："时间计算器"

## 4. 功能逻辑

### 4.1 预设时间管理
- **增加：** 用户点击"增加"→ 弹出滚轮选择器 → 选择时分 → 点击确定 → 时间保存到本地列表
- **删除：** 左滑或长按列表项 → 显示删除按钮 → 点击删除 → 从本地列表移除
- **选中：** 点击列表项 → 该项高亮 → 标记为当前选中时间
- **数据键名：** `presetTimes`，存储为 JSON 数组 `[{hour: 8, minute: 30}, ...]`

### 4.2 时间计算逻辑
```
当前时间：2026-04-20 14:30
用户选中：08:00

计算目标时间：2026-04-21 08:00:00
时长 = 目标时间 - 当前时间
= 17小时30分钟
```

特殊情况：
- 若明天的目标时间已过（如当前 10:00，选了 08:00），结果提示"已过该时间"
- 跨天、跨月（如2月最后一天到3月1日）均正确处理

### 4.3 数据持久化
- 所有预设时间存储在小程序本地 storage
- key: `presetTimes`，value: JSON 数组
- 页面加载时读取，保存时写入

## 5. 页面状态

| 状态 | 描述 |
|------|------|
| 初始空状态 | 列表为空，显示空状态提示，"计算"按钮不可用 |
| 有预设时间 | 列表显示时间，可选中，"计算"按钮可选 |
| 选择器弹窗 | 底部弹窗显示滚轮选择器 |
| 已选中待计算 | 列表项高亮，点击"计算"显示结果 |
| 显示结果 | 结果区域展示时长，可点击左上角返回 |

## 7. 部署与上架

微信小程序上架流程：
1. 注册微信小程序账号（https://mp.weixin.qq.com/）
2. 下载微信开发者工具
3. 在开发者工具中创建项目，填入 AppID
4. 开发调试（可在开发者工具中模拟）
5. 功能完成且测试通过后，在开发者工具中上传代码
6. 在微信公众平台提交审核（通常1-7个工作日）
7. 审核通过后自动上线

**注意：** 个人主体小程序可上架大部分类目，但部分类目（如医疗、金融）需要特殊资质。
